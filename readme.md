
## ğŸ¨ Design System

### Colors
- Background: `#000000` (Black)
- Primary: `#ff0000` (Red)
- Text: `#ffffff` (White)
- Accent: `#cc0000` (Dark Red)

### Typography
- Titles: "Rubik 80s Fade"
- Body: "Bebas Neue"

### Spacing Scale
- XS: 0.25rem
- SM: 0.5rem
- MD: 1rem
- LG: 2rem
- XL: 3rem

### Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px
